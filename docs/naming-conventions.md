# Naming conventions

Recommended file naming for SharePoint/Teams storage:

- Agendas: YYYY-MM-DD_agenda_meeting-name.md
- Minutes: YYYY-MM-DD_minutes_meeting-name.md
- Actions export: YYYY-MM-DD_action-log.md

Principles:
- Date first for easy sorting
- Use short, consistent meeting names
- Keep one source of truth in SharePoint
